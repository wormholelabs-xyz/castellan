# Egress allow-list for the castellan DNS firewall.
#
# Each line's type is determined by its prefix (checked exact -> suffix -> wildcard -> regex):
#
#   api.anthropic.com                              exact host (case-insensitive)
#   .sentry.io                                     suffix: matches sentry.io AND any subdomain *.sentry.io
#   *.gallery.vsassets.io                          wildcard: any subdomain (one or more labels) of the suffix
#   re:^[a-z0-9-]+\.pkg\.dev$                      regex matched against the whole lowercased name
#   JsonList(https://example.com/ips, [key1, key2])        fetch JSON; keys are top-level string arrays
#   JsonList(https://example.com/ips, [arr[].field])        fetch JSON; keys traverse array-of-objects
#
# Blank lines and everything after `#` are ignored.
